# Kagi-Kashi（笑）

アプリ制作プロジェクトのやつ

- 縦並びに表示。
- 開錠。施錠。一旦施錠している場合、メッセージを残せる。
- 部室を持っているサークルを上から表示させる。
- お気に入り機能。
- 通知
- 名前検索。
- 活動報告書。

## 手順

1. ログイン
2. 部活動一覧が出ている。（一時施錠：赤、施錠中：灰色、活動中：緑）
3. 部活動をクリック
4. 部活動詳細ページに飛ぶ
5. 活動報告書、状況、鍵の貸し出しが選択できる画面がある。

## 鍵を借りる場合

- その部活に学籍番号が登録されている。
- 学籍番号、名前、鍵番号を入力後借りれる。
